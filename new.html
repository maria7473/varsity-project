Vanilla JavaScript

Vanilla JavaScript is a term used to refer to JavaScript code that is written without the aid of any external libraries or frameworks. It is simply a pure JavaScript code written in its raw form, without any plugins or other libraries.

React

React is a web application library (not a framework, we'll get to that later why) developed by Facebook and open-sourced in 2013. It uses JavaScript as its primary language that it is written in,(TypeScript is used too, but it compiles to JavaScript so anyways yeah).

Node.js

Node.js is an open-source and cross-platform JavaScript runtime environment. It is a popular tool for almost any kind of project! Node.js runs the V8 JavaScript engine, the core of Google Chrome, outside of the browser. This allows Node.js to be very performant.

MongoDB
